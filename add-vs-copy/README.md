#This will gives the difference between COPY and ADD.

##Docker COPY command Syntax
`COPY <src> <dst>`

`<src>: Source directory/file`
`<dst>: Destination directory/file`

##Docker ADD command Syntax
`ADD <src> <dst>`

`<src>: Source file/directory`
`<dst>: Destination`
